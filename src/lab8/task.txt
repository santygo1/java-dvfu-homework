Разработать класс-дженерик для хранения данных.
Данные должны храниться в поле-массиве, ограниченном максимальным размером (например, 10 элементов).
Реализовать методы добавления данных и удаления данных (например, по принципу стека).
Методы должны генерировать исключение при попытке добавить данное в уже
заполненный контейнер и удалить из пустого контейнера. Для этого создать свои классы исключений.
Кроме этого реализовать поиск элементов.пп Метод генерирует исключение, если контейнер пуст.
